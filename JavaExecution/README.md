JavaExecution
==========

概要
-----------
Jarをダブルクリックで起動できない人向けの**アプリケーション**です。  

使い方
-----------
`.jar`とアプリを紐付けることで、Jarファイルを実行できるようにします。  
ダブルクリックでJarファイルが起動しない不具合に悩まされている人向けです。  
(実はレジストリのコマンド設定で解決できたりします。)
