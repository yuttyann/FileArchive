OpenInventory
==========

## 留意事項
古い仕様のプラグインであるため、基本的に修正や機能追加等のアップデートは行いません。  
また、最新のバージョンの情報を記載しています。

概要
-----------
プレイヤーのインベントリを遠隔で開き操作する事ができる**Bukkitプラグイン**です。  

コマンド
-----------
<details>
<summary>openinventory / opinv</summary>

| 名称 | 短縮 |
|:---|:---|
| openinventory | opinv |

| 引数 | 権限 | 初期 | 説明 |
|:---|:---|:---|:---|
| [player] | yp.opinv | OP | 指定したプレイヤーのインベントリを開きます。 |
</details>
